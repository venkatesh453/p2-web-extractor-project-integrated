
# Web Page Extractor (Fully Integrated)

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Start the project (backend + frontend in browser):
   python start.py

Visit: http://127.0.0.1:8000

You can enter any URL to extract:
- Full text
- Anchor links
- Images

Data is saved into: data.db (SQLite)
